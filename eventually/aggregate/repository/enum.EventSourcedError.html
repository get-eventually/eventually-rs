Enum eventually::aggregate::repository::EventSourcedError

pub enum EventSourcedError<E, SE, AE> {
    RehydrateAggregate(E),
    StreamFromStore(SE),
    AppendToStore(AE),
}
Expand description

List of possible errors that can be returned by an EventSourced method.

Variants§

§

RehydrateAggregate(E)

This error is returned by EventSourced::get when the desired Aggregate returns an error while applying a Domain Event from the Event Store during the rehydration phase.

This usually implies the Event Stream for the Aggregate contains corrupted or unexpected data.

§

StreamFromStore(SE)

This error is returned by EventSourced::get when the Event Store used by the Repository returns an unexpected error while streaming back the Aggregate’s Event Stream.

§

AppendToStore(AE)

This error is returned by EventSourced::store when the Event Store used by the Repository returns an error while saving the uncommitted Domain Events to the Aggregate’s Event Stream.
