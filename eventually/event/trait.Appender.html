Expand description

Interface used to append new Domain Events in an Event Store.

Required Associated Types§

source

type Error: Into<Option<ConflictError>> + Send + Sync

The error type returned by the Store during an [append] call. It could be a [version::ConflictError], which is why the bound to Into<Option<ConflictError>> is required.

Appens new Domain Events to the specified Event Stream.

The result of this operation is the new Version of the Event Stream with the specified Domain Events added to it.
