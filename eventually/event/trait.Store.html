Expand description

An [Event] Store, used to store Domain Events in Event Streams – a stream of Domain Events – and retrieve them.

Each Event Stream is represented by a unique Store::StreamId.

type AppendError: Into<Option<ConflictError>> + Send + Sync

The error type returned by the Store during an [append] call. It could be a [version::ConflictError], which is why the bound to Into<Option<ConflictError>> is required.
