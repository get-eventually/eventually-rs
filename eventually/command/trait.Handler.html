Expand description

A software component that is able to handle Commands of a certain type, and mutate the state as a result of the command handling, or fail.

In an event-sourced system, the Command Handler should use an Aggregate to evaluate a Command to ensure business invariants are respected.

Handles a Command and returns an error if the handling has failed.

Since Commands are solely modifying the state of the system, they do not return anything to the caller but the result of the operation (expressed by a Result type).
