Expand description

An AggregateRoot represents an handler to the Aggregate it’s managing, such as:

  • Owning its State, Id and version,
  • Proxying Commands to the Aggregate using the current State,
  • Keeping a list of Events to commit after Command execution.

Initialize

An AggregateRoot can only be initialized using the AggregateRootFactory.

Check AggregateRootFactory::build for more information.
